# XPBD Cloth Simulation

Cloth simulation based on XPBD paper.
Made with:
- C++
- DirectX
- ImGui

Steering:
- Press mouse wheel to rotate around the center.
- Scroll to zoom in/out

Version 1.0 video demo: https://www.youtube.com/watch?v=LXo_VOUzQ9o
